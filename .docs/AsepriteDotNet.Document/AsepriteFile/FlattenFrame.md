# AsepriteFile.FlattenFrame method (1 of 2)

Flattens the specified [`Frame`](../Frame.md) by blending each [`Cel`](../Cel.md) in the [`Frame`](../Frame.md), starting with the top most cel and blending down. The result is an array of Color elements representing the final flattened image of the frame.

```csharp
public Color[] FlattenFrame(Frame frame, bool onlyVisibleLayers)
```

| parameter | description |
| --- | --- |
| frame | The [`Frame`](../Frame.md) to flatten |
| onlyVisibleLayers | Whether only the [`Cel`](../Cel.md) elements that are on a [`Layer`](../Layer.md) that is visible should be included. |

## Return Value

A new Array of Color elements that represent the image of the [`Frame`](../Frame.md) once it has been flattened.

## Remarks

Any instance of a [`TilemapCel`](../TilemapCel.md) within the [`Frame`](../Frame.md) is ignored.
