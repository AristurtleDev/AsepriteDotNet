# AsepriteFile.FlattenFrame method (1 of 2)

Flattens the specified [`Frame`](../FrameType) by blending each [`Cel`](../CelType) in the [`Frame`](../FrameType), starting with the top most cel and blending down. The result is an array of Color elements representing the final flattened image of the frame.

```csharp
public Color[] FlattenFrame(Frame frame, bool onlyVisibleLayers)
```

| parameter | description |
| --- | --- |
| frame | The [`Frame`](../FrameType) to flatten |
| onlyVisibleLayers | Whether only the [`Cel`](../CelType) elements that are on a [`Layer`](../LayerType) that is visible should be included. |

## Return Value

A new Array of Color elements that represent the image of the [`Frame`](../FrameType) once it has been flattened.

## Remarks

Any instance of a [`TilemapCel`](../TilemapCelType) within the [`Frame`](../FrameType) is ignored.
